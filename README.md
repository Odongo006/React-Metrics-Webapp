# 📖 Metrics webapp<a name="about-project"></a> <!---Project title-->
This React capstone project is about building a mobile web application to check a list of metrics that We've created making use of React and Redux. This website list the genres of popular shows and allows to see the shows descriptions.



## 🛠 Built With <a name="built-with"></a>
 React, Redux and CSS, and linter configuration setup.

## 💻 Getting Started <a name="getting-started"></a>

- Create a local directory that you want to clone the repository.

- Open the command prompt in the created directory.

- On the terminal run this command git clone [https://github.com/Odongo006/React-Metrics-Webapp]

- Go to the repository folder using command prompt cd React-Metrics-Webapp


- Install the dev dependencies for linters run npm install.

- To start run npm start

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Prerequisites

In order to run this project you need:
